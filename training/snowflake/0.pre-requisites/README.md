# Pre-requisites

Before starting the training, complete the two setup steps below.

| Step | File | Description |
|------|------|-------------|
| 1 | [Setup Account](setup%20account.md) | Create a Snowflake trial account and configure the initial settings |
| 2 | [Python Environment (uv)](uv.md) | Install uv, sync the project dependencies, and verify your installation |

Both steps are required. Start with **Setup Account** and then follow **uv**.
